# HireWhen

HireWhen is a static HTML, CSS, and JavaScript site for small-business hiring calculators.

## Project Structure

- `index.html`: main "Can you afford to hire?" calculator
- `revenue-needed-to-hire.html`: reverse calculator for required monthly revenue
- `main.js`: controller for the main calculator page
- `reverse-calculator.js`: controller for the reverse calculator page
- `calculator-utils.js`: shared calculator math, formatting, insights, and summary helpers
- `style.css`: shared site styles

## Notes

- The site is fully static.
- There is no backend.
- There are no runtime dependencies.
- No build step is required.

Open `index.html` or `revenue-needed-to-hire.html` directly in a browser, or serve the folder with any static file host.
